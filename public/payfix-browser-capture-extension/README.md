# PayFix Page Capture

Use this extension when a docs/download portal requires your browser login.

## Chrome / Microsoft Edge

1. Open `chrome://extensions` or `edge://extensions`.
2. Enable `Developer mode`.
3. Click `Load unpacked`.
4. Select this folder:
   `public/payfix-browser-capture-extension`
5. Open the logged-in page you want PayFix to inspect.
6. Click the PayFix Page Capture extension.
7. Click `Share visible page`.
8. Return to PayFix and click `Import shared page`.

The extension sends only the page you explicitly share: URL, title, visible text, selected text, and visible links.
